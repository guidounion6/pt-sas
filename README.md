## Getting Started

Primer paso: 

instalar: npm install 

o su derivado correspondiente ya sea yarn o pnpm

Segundo paso: 

Uan vez instalado se correra el programa con: 

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Se ejecuta en [http://localhost:3000](http://localhost:3000)
